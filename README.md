# Модуль автоматизации учёта клиентов малого бизнеса

## Описание
Консольный программный модуль на Python для ввода, хранения, поиска и генерации отчётов по клиентской базе. База данных — PostgreSQL.

## Структура проекта
```
project/
├── config.py       — настройки подключения к БД
├── db.py           — функции работы с БД (CRUD, отчёты)
├── main.py         — главное меню (точка входа)
├── test_db.py      — модульные тесты
├── create_db.sql   — SQL-скрипт создания таблиц
└── README.md       — данный файл
```

## Установка и запуск

### 1. Установить зависимости
```bash
pip install psycopg2-binary
```

### 2. Создать базу данных
```bash
psql -U postgres -c "CREATE DATABASE clients_db;"
```

### 3. Настроить подключение
Отредактируйте `config.py` — укажите свои `user`, `password`, `host`, `port`.

### 4. Запустить программу
```bash
python main.py
```
Таблицы создадутся автоматически при первом запуске.

### 5. Запустить тесты
```bash
python -m unittest test_db.py -v
```

## Функциональность
- Добавление / просмотр / поиск / редактирование / удаление клиентов
- Учёт заказов клиентов (добавление, просмотр, смена статуса)
- Отчёт по дате регистрации клиентов
- Сводный отчёт (кол-во клиентов, заказов, общая сумма)
- Обработка ошибок ввода
